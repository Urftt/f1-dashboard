1. In `frontend/src/types/session.ts`, add the `StandingRow` interface:
```typescript
export interface StandingRow {
  driver: string           // abbreviation (e.g., "VER")
  fullName: string         // from DriverInfo
  teamColor: string        // hex from DriverInfo
  position: number
  prevPosition: number | null  // position at currentLap - 1 (null at lap 1)
  gap: number | null       // seconds behind leader (null for leader)
  interval: number | null  // seconds behind car ahead (null for leader)
  isLapped: boolean
  lapsDown: number         // 0 = lead lap, 1 = +1 LAP, etc.
  compound: string | null  // 'SOFT' | 'MEDIUM' | 'HARD' | 'INTERMEDIATE' | 'WET'
  tyreLife: number | null
  pitStops: number
  compoundChanged: boolean // true if compound differs from previous lap
}
```

2. Create `frontend/src/components/StandingsBoard/useStandingsData.ts` with:

- Export `COMPOUND_DISPLAY` constant map: `{ SOFT: { letter: 'S', color: '#E8002D' }, MEDIUM: { letter: 'M', color: '#FFF200' }, HARD: { letter: 'H', color: '#FFFFFF' }, INTERMEDIATE: { letter: 'I', color: '#39B54A' }, WET: { letter: 'W', color: '#0067FF' } }`

- Export `useStandingsData()` hook that:
  a. Reads `laps`, `drivers`, `currentLap` from `useSessionStore` (same selector pattern as useGapData.ts)
  b. Returns `useMemo(() => { ... }, [laps, drivers, currentLap])` computing `StandingRow[]`

- Inside the useMemo:
  a. Build `driverMap` from `drivers` array: `new Map(drivers.map(d => [d.abbreviation, d]))`
  b. Filter `currentLapRows = laps.filter(l => l.LapNumber === currentLap)` — one row per driver
  c. Filter `prevLapRows = currentLap > 1 ? laps.filter(l => l.LapNumber === currentLap - 1) : []` — for position delta
  d. Build `prevPositionMap = new Map(prevLapRows.map(l => [l.Driver, l.Position]))` for O(1) lookup
  e. Sort currentLapRows by `(a.Position ?? 99) - (b.Position ?? 99)` — null positions go to bottom
  f. Determine leader: first row after sort. `leaderTime = leader.Time`
  g. Detect lapped cars: for each driver, find their max LapNumber up to currentLap. Compare against leader's max LapNumber. `lapsDown = leaderMaxLap - driverMaxLap`. This requires a pre-computed map:
     ```
     const maxLapByDriver = new Map<string, number>()
     for (const l of laps) {
       if (l.LapNumber !== null && l.LapNumber <= currentLap) {
         const prev = maxLapByDriver.get(l.Driver) ?? 0
         if (l.LapNumber > prev) maxLapByDriver.set(l.Driver, l.LapNumber)
       }
     }
     const leaderMaxLap = maxLapByDriver.get(sortedRows[0]?.Driver ?? '') ?? currentLap
     ```
  h. Compute pit stops per driver: count laps where `PitInTime !== null` up to currentLap:
     ```
     const pitCountMap = new Map<string, number>()
     for (const l of laps) {
       if (l.LapNumber !== null && l.LapNumber <= currentLap && l.PitInTime !== null) {
         pitCountMap.set(l.Driver, (pitCountMap.get(l.Driver) ?? 0) + 1)
       }
     }
     ```
  i. Detect compound changes: build map of compound at previous lap:
     ```
     const prevCompoundMap = new Map(prevLapRows.map(l => [l.Driver, l.Compound]))
     ```
  j. Map each sorted row to StandingRow:
     - `driver`: row.Driver
     - `fullName`: driverMap.get(row.Driver)?.fullName ?? row.Driver
     - `teamColor`: driverMap.get(row.Driver)?.teamColor ?? '#888888'
     - `position`: row.Position ?? 99
     - `prevPosition`: prevPositionMap.get(row.Driver) ?? null
     - `gap`: compute only if not leader AND `lapsDown === 0` AND `leaderTime !== null` AND `row.Time !== null`: `row.Time - leaderTime`. Otherwise null.
     - `interval`: for position > 1 and `lapsDown === 0`, compute `row.Time - carAheadTime` where carAhead is the previous row in sorted order with non-null Time. Otherwise null.
     - `isLapped`: lapsDown > 0
     - `lapsDown`: from maxLapByDriver comparison
     - `compound`: row.Compound
     - `tyreLife`: row.TyreLife
     - `pitStops`: pitCountMap.get(row.Driver) ?? 0
     - `compoundChanged`: `currentLap > 1 && row.Compound !== null && prevCompoundMap.get(row.Driver) !== row.Compound && prevCompoundMap.get(row.Driver) !== undefined`. Set to `false` when `currentLap === 1` to avoid false positives at race start.

  k. Return the StandingRow[] array.

IMPORTANT: Use `LapRow.Time` (session elapsed time) for gap/interval calculation, NOT `LapTime`. This is a known pitfall documented in STATE.md.
IMPORTANT: `Position` can be null — always use `?? 99` when sorting. Filter out rows with null Position from gap calculations but still include them at the bottom of the table.

   - If compound is null: render "---"

**Step 1: Create frontend/src/components/StintTimeline/StintTimeline.tsx**

- Use the CJS interop pattern for react-plotly.js import (same as GapChart.tsx)

- If traces is empty, render a muted "No stint data available" placeholder div
- Build Plotly layout:

  - `height`: 500 (per decision: ~500px so all 20 drivers fit at ~20-24px per row)
  - `showlegend: false`
  - `hovermode: 'closest'`
  - `bargap: 0.15` — small gap between driver rows for readability
  - `shapes`: filter null from cursorShapes array
- Build Plotly config: `{ responsive: true, displayModeBar: false }`
- Render `<Plot data={traces} layout={layout} config={config} className="w-full" />`
